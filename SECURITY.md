# Security Notice

This is a submission-only repository for the Google for Startups AI Agents Challenge 2026.

## What's Included
- Architecture documentation and diagrams
- Sanitized data model definitions (Pydantic schemas)
- API route overview (endpoints, not implementation)
- Voice tool catalog (tool names and purposes, not prompts)
- Track 3 refactor specification
- Technical findings and learnings

## What's NOT Included (by design)
- Production source code (serve.py, routes, services)
- Voice persona prompts (elder, demo, caregiver, dev, feedback)
- Firestore credentials or project configuration
- Stripe API keys or billing secrets
- Demo codes or access credentials
- Customer data or Firestore exports
- Pricing configuration internals
- Docker/deployment scripts
- Frontend source code

## Why
The competitive advantage of Vita is not in the tech stack (FastAPI + Gemini + Firestore is public knowledge). It's in the cultural specificity of the voice personas, the family architecture design, the neuroscience grounding, and the accumulated elder data (Libro de Vida). These are not shareable and are not necessary to evaluate the technical implementation.
